A new way to write digitally, a interesting way to read, ranking of stories by number of comments and in the end, a beautiful designed social network. This is how I'll describe my latest project — Markdawn. If you're looking for a place to store and share your Markdown articles, this is it!

  • Type Writing platform made for Markdown syntax.
  • Technology Django + PostgreSQL + Ubuntu Server + Sass
  • Link markdawn.com
  • Status Put on hold due to low usage.
