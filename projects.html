Markdawn

2013 Publishing Platform
A new way to write digitally, a interesting way to read, ranking of stories by number of comments and in the end, a beautiful designed social network. This is how I'll describe my latest project — Markdawn. If you're looking for a place to store and share your Markdown articles, this is it!

Moss

2012 OS Experiment
Moss wants to be an experimental OS that can handle new technologies and promise to be on desktop as simple as iOS is on mobile. Everybody deserves power and simplicity on a big monitor screen.

Peculiar

2010 CSS Icons
Peculiar is a free icon package made only in CSS. It was created for sites and web applications that depend on fewer HTTP requests as possible or don't need to use any image at all. The package contains 45 pictograms.

The Journalist

2007 WordPress theme
The Journalist, or redefinition of minimalist, is a smart, sophisticated WordPress theme designed for professional journalists.
