Lira Media is an unique concept of email marketing. Designed not only to delight the receiver of promotions, but also the marketer.

The main responsibilities for the project were: design direction, interface, workflows, implementation of the layout using latest web technologies. The responsive design was created for a few Django powered web apps.

Lira Apps Suite

A complex suite of web apps: Analytics, Offers, Scheduler, Delivery doing one thing and one thing only: sending email campaigns.

Lira Analytics was used to determine what was the best campaign for our clients.

Lira Offers was the management interface.

Lira Analytics was enriched with beautiful charts.

Lira Delivery was used for sending email campaigns using specialized burst mode.

Lira Delivery was also used for network infrastructure.

Lira Delivery was as user friendly as the rest of the apps.
